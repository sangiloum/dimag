# dimag
Wordpress theme for a small research group. 

Here is a list of plugins recommended to be installed.

## Must install for the list of papers / talks
- Pods - Custom Content Types and Fields

Here's a configuration for Pods. In "Pods Admin", active the component called "Migrate: Packages". Then use it to import the following configuration to activate the following 4 custom types.

* paper_type (Paper Types)
* paper (Papers)
* researcher (Researchers)
* talk (Talks)
